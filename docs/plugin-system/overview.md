# 插件系统概述

KnowFlow 插件系统支持动态扩展功能。

---

## 设计目标

| 目标 | 说明 |
|------|------|
| 动态加载/卸载 | 插件可以在运行时动态加载和卸载 |
| 标准化接口 | 定义统一的插件接口，确保兼容性 |
| 前后端分离 | 插件可以同时提供后端 API 和前端组件 |
| Key 注册 | 插件可以注册自定义 Key 定义 |

---

## 插件能力

| 能力 | 说明 |
|------|------|
| 注册 Key | 插件可以定义自定义 Key，存储到数据库 |
| 后端 API | 插件可以注册自定义 FastAPI 路由 |
| 前端组件 | 插件可以提供 React 组件，动态渲染 |
| 生命周期 | 支持 `on_load` 和 `on_unload` 钩子 |

---

## 插件类型

| 类型 | 说明 |
|------|------|
| 知识源导入 | 从外部系统导入知识项 |
| 检索插件 | 自定义检索算法 |
| 知识处理 | 对知识项进行处理 |
| 可视化 | 自定义可视化展示 |
| 自动化 | 自动化任务 |
